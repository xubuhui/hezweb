## 1. 系统概述

HEZ 记账系统是一款功能强大的微信记账管理工具，集成了以下核心功能：

- **微信账号管理** - 管理微信登录状态，监控微信进程
- **群聊记账** - 支持多群聊独立记账，支持卡密交易
- **群聊分组** - 对群聊进行分类管理
- **自动回复** - 智能关键词自动回复系统
- **群组群发** - 批量发送消息到多个群聊
- **员工记录** - 查看员工操作记录
- **数据统计** - 可视化数据分析和报表
- **卡密分析** - 卡密交易盈亏分析
- **指令管理** - 自定义群聊命令
- **授权管理** - 管理员权限管理

## 3. 安装与启动

### 3.1 使用打包版（推荐）

1. 下载 `web_app.exe`
2. 双击运行 `web_app.exe`
3. 系统会自动启动并打开浏览器登录页面

### 3.2 使用源代码

1. 确保已安装 Python 3.12 或更高版本
2. 安装依赖：
   ```powershell
   pip install -r requirements.txt
   ```
3. 启动系统：
   ```powershell
   python web_app.py
   ```

### 3.3 系统目录结构

```
vx/
├── backend/           # 后端代码
├── data/              # 数据目录（数据库文件）
├── logs/              # 日志目录
├── static/            # 静态资源
├── templates/         # 网页模板
├── Weixin/            # 微信文件目录
├── web_app.py         # 主程序
└── requirements.txt   # 依赖列表
```

## 5. 功能模块

### 5.1 账号管理

**功能描述**：管理微信账号的启动、停止和状态监控

**主要操作**：
- **添加账号**：配置微信路径并启动微信
- **启动微信**：启动微信进程
- **停止微信**：停止微信进程
- **刷新状态**：刷新微信在线状态

**注意事项**：
- 系统支持单账号模式，同时只能运行一个微信实例
- 确保 Weixin 目录下有正确的微信文件

---

### 5.2 群聊分组

**功能描述**：对群聊进行分类管理，便于批量操作和统计

**默认分组**：
- 收卡群（绿色）
- 出卡群（蓝色）
- 内部群（橙色）

**主要操作**：
- **新建分组**：创建新的群聊分组
- **添加群聊**：将群聊添加到指定分组
- **移除群聊**：将群聊从分组中移除
- **同步群名**：同步更新群聊名称

---

### 5.3 自动回复

**功能描述**：根据关键词自动回复消息

**规则配置**：
- **触发类型**：关键词、正则表达式
- **匹配方式**：包含、精确匹配、正则匹配
- **回复类型**：文本消息
- **作用范围**：全部群聊、指定群聊
- **冷却时间**：防止频繁触发
- **时间范围**：指定时间段生效
- **星期限制**：指定星期几生效

**使用步骤**：
1. 点击「新建规则」
2. 填写规则名称和触发条件
3. 设置回复内容
4. 配置作用范围和时间限制
5. 保存并启用规则

---

### 5.4 群组群发

**功能描述**：批量发送消息到多个群聊

**安全提示**：
- 发送间隔建议 ≥ 3 秒
- 内容需健康合规
- 系统会在间隔基础上增加随机时间

**使用步骤**：
1. 点击「新建群发」
2. 选择目标群聊（可按分组选择）
3. 输入发送内容
4. 设置发送间隔
5. 点击「开始发送」

**进度监控**：
- 实时显示发送进度
- 记录成功和失败数量
- 支持中途中止任务

## 7. 记账格式

### 7.1 简单格式

| 格式 | 说明 | 示例 |
|------|------|------|
| `+金额` | 收入 | `+600` → 收款 600 |
| `-金额` | 支出 | `-600` → 付款 600 |
| `+金额*数量` | 收入（乘法） | `+600*4` → 收款 600×4=2400 |
| `-金额*数量` | 支出（乘法） | `-600*4` → 付款 600×4=2400 |

### 7.2 带备注格式

在金额后面加空格和备注文字：

```
+600 货款
-500 运费
+600*4 批量收款
```

### 7.3 卡密交易格式

**格式**：`+面值*汇率 卡密 [备注]`

**示例**：
```
+500*6 A7S9K2P8R5T4Q1X6
+500*6 ABCD 1234 WXYZ 老客户
```

**字段说明**：
- `面值`：卡密面值
- `汇率`：兑换汇率
- `卡密`：卡密内容（支持多个词，自动识别）
- `备注`：可选备注信息

**卡密格式要求**：
- 只包含字母、数字、空格、-
- 长度 8-25 字符

### 7.4 表达式计算

支持简单的数学表达式：

```
+600+200
-1000-200
+500*2+300
```

支持的运算符：`+`、`-`、`*`、`×`、`/`、`÷`

## 8. 常见问题

### 8.1 登录问题

**Q: 登录失败怎么办？**
A: 请检查用户名和密码是否正确，确保网络连接正常。

**Q: 授权到期了怎么办？**
A: 请联系管理员续期授权。

---

### 8.2 微信相关问题

**Q: 微信无法启动？**
A: 请检查 Weixin 目录下的文件是否完整，确认微信版本为 4.1.8.28。

**Q: 微信掉线了怎么办？**
A: 在「账号管理」页面点击「刷新状态」或重新启动微信。

---

### 8.3 记账问题

**Q: 发送记账消息没有反应？**
A: 请确认该群聊已开启记账功能（发送 `/开启`）。

**Q: 如何撤回错误的记账？**
A: 在群聊中发送 `/撤回` 命令。

**Q: 如何清空所有记录？**
A: 在群聊中发送 `/清账` 命令（需要管理员权限）。

---

### 8.4 数据安全

**Q: 数据存储在哪里？**
A: 所有数据存储在本地 `data/accounting.db` SQLite 数据库文件中。

**Q: 如何备份数据？**
A: 定期备份 `data/` 目录和 `logs/` 目录即可。

**Q: 系统会上传数据吗？**
A: 系统仅在授权验证时连接服务器，所有业务数据存储在本地。

---

### 8.5 自动回复问题

**Q: 自动回复没有触发？**
A: 请检查：
1. 规则是否已启用
2. 触发条件是否匹配
3. 是否在作用范围内
4. 是否在冷却时间内
5. 是否在指定时间范围内
