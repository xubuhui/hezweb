第一章

快速入门

1

软件安装与登录

从客服处获取最新版安装包，根据系统版本选择对应安装包（目前只支持windows系统）。

安装完成后，账号密码登录软件，使用授权码完成激活。

登录成功后，系统会自动进行环境检测，确保运行环境正常。

$ hez --activate YOUR_LICENSE_KEY
[INFO] 正在验证授权码...
[OK] 授权验证成功
[INFO] 设备指纹: xxxxxxxx
[INFO] 授权到期: 2027-03-28
[OK] 环境检测通过，欢迎使用 HEZ
2

首次配置

扫码登录微信

将机器人微信拉进需要记账的群聊

添加需要使用机器人记账的员工微信好友

3

开始使用

员工微信在任意微信群发送/授权码，机器人微信会反馈员工微信id，然后去软件上员工列表开启管理员权限

有管理员权限的员工可以在群聊发送/开启收货群 /开启出货群 /开启内部群 指令 就可以记账和群发了

建议按业务分别开启收货/出货群聊 方便后台交易的数据分析

第二章

授权管理

权限体系（目前只做了管理员权限，后续优化）

系统采用多级权限管理，确保操作安全可控：

超级管理员

拥有全部权限，可管理所有群和员工

群管理员

管理指定群聊，可分配员工权限

记账员

仅可进行记账操作，无管理权限

查看员

仅可查看数据，无操作权限

员工微信授权流程

员工微信使用需完成以下步骤：

1

员工发送授权码到群聊

员工在群聊中发送"授权码"，机器人自动反馈该员工的wxid

2

后台审核并开启授权

管理员在后台审核员工wxid，确认无误后开启授权

3

授权成功，员工可操作系统功能

根据分配的权限范围，员工可正常操作对应的功能

第三章

记账功能

支持的记账格式

系统支持多种记账格式，灵活满足不同业务场景：

基础格式
+600      # 收款600元
−600      # 付款600元
带乘数格式（面值×汇款）
+600*4    # 600×4 = 2400
−600*4
带乘数和卡密格式
+600*4 卡密   # 600×4 =2400 附带卡密信息
−600*4 卡密
双乘数格式（面值×倍数×汇率）
+500*5*5  # 500×5×5 = 12500
−500*5*5
带除数格式（金额÷汇率）
+800/4    # 800÷4 = 200
−600/4
混合运算格式（金额÷汇率+加数）
+800/4+8  # 800÷4+8 = 208
−600/4+9
使用说明
  • 乘法运算符使用 *，除法运算符使用 /
  • 支持混合运算，系统会按从左到右顺序计算
  • 卡密信息可选，用于记录交易凭证

多群账本管理

每个群聊拥有独立账本，数据互不干扰。

支持跨群汇总统计，方便全局把控。

群类型说明

不同群类型对应不同的记账规则：

群类型 说明 记账规则

群发功能

支持按分组群发消息，可设置发送间隔防封：

指令格式：/群发 分组 间隔 内容

示例：
/群发 分组1 5s 今日汇率已更新
/群发 分组2 3s 新卡到货，欢迎选购

第六章

卡密管理

卡密记账规则

卡密交易是核心业务，需要详细记录以下信息：

格式：+面值*汇率 卡密

示例：
+500*6 xxxxxxxx   # 面值500，汇率6，卡密xxxxxxxx
+100*7.2 yyyyyyyy # 面值100，汇率7.2，卡密yyyyyyyy
+50*1.1 zzzzzzzz  # 面值50，汇率1.1，卡密zzzzzzzz

说明：
- 面值：卡的原始金额
- 汇率：兑换人民币的汇率
- 卡密：卡密的标识或后四位

汇率计算规则

多条汇率记录时的计算方式：

重要规则
汇率连乘取最后一条记录的值。例如：消息中包含多个*数字，系统会取最后一个*后面的数字作为汇率。 
示例解析：
消息："+500*6.1*6.2 xxx eur"
→ 汇率取 6.2（最后一个）
→ 实际金额 = 500× 6.1× 6.2   = 18,910 元

HEZ 内置多协议防封引擎，支持动态切换登录协议：

安全建议

避免在短时间内大量添加好友或频繁加群，控制操作频率。

第十章

常见问题

登录失败怎么办？
解决方案
1. 确认账号密码是否正确，注意大小写
2. 检查网络连接是否稳定
3. 如忘记密码，请联系客服
如何更换设备？

授权码为一次性绑定，绑定后不可更换设备。如需更换设备，请联系客服处理：

更换设备流程
1. 提供原设备的设备信息
2. 说明更换设备的原因
3. 客服审核通过后生成新授权码
账单数据丢失如何恢复？

建议定期导出账单：

数据保护建议
1. 每周定期导出数据
2. 如果没有备份只能手动重新记账
收不到记账消息怎么办？

请按以下步骤排查：

1. 确认机器人是否已成功加入群聊
2. 检查群聊是否开启了记账功能
3. 检测是否开启了管理员权限
3. 确认记账格式是否符合规范
4. 查看是否被移出了群聊 5. 查看网络是否通畅

第十一章

故障排除

常见错误代码

错误代码 说明 解决方案
ERR_001 授权码无效 联系客服重新获取授权码
ERR_002 设备绑定失败 当前设备已绑定其他账号
ERR_003 授权已过期 续费后重新激活
ERR_004 网络连接失败 检查网络或切换网络环境
ERR_005 协议连接异常 尝试切换其他协议
